# Relaciones de 02_titulo_i_artistas_interpretes_o_ejecutantes_parcial

## Función del bloque

Este fragmento abre el Título I del Libro Segundo y presenta el arranque del estatuto jurídico de artistas intérpretes o ejecutantes.

## Relaciones directas

- Se relaciona con el bloque previo de obras audiovisuales porque muchas interpretaciones se incorporan a fijaciones audiovisuales.
- Se relaciona con los bloques de fonogramas y grabaciones audiovisuales porque la actuación del intérprete es el presupuesto de la fijación posterior.
- Se relaciona con radiodifusión porque la comunicación pública y difusión de actuaciones puede hacerse a través de emisiones.

## Enlaces conceptuales útiles

- Artista intérprete o ejecutante -> tiene derecho a -> fijación.
- Artista intérprete o ejecutante -> tiene derecho a -> reproducción.
- Artista intérprete o ejecutante -> tiene derecho a -> comunicación pública.
- Comunicación pública -> puede generar -> remuneración equitativa.
- Cesión al productor -> no elimina siempre -> derecho irrenunciable de remuneración.
